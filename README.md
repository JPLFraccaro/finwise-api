# Finwise API

## 🇬🇧 English

📦 Personal Financial Management (PFM) RESTful API  

Finwise is a modular RESTful API for personal finance management. It allows users to record income, expenses, budgets, and savings goals, offering comprehensive and well-documented endpoints.  

This repository is part of the *Fintech AI Pack* project, designed to demonstrate strong backend development capabilities and readiness for financial SaaS applications.  

### Features

- Income and expense tracking  
- Customizable tags and categories  
- Savings goals and budgets
- Basic API reporting  
- Simple JWT authentication

## 🇦🇷 Español

📦 API REST para gestión de finanzas personales (PFM)

Finwise es una API REST modular para el manejo de finanzas personales. Permite registrar ingresos, gastos, presupuestos y objetivos de ahorro, ofreciendo endpoints completos y documentados.

Este repositorio forma parte del proyecto *Fintech AI Pack*, diseñado para demostrar capacidades sólidas en desarrollo backend y preparación para aplicaciones SaaS financieras.

### Características

- Registro de ingresos y gastos
- Etiquetas y categorías personalizadas
- Objetivos financieros y presupuestos
- Reportes básicos por API
- Autenticación JWT simple
